# PentesterFlow — Marketing site

A small, fast marketing site for [PentesterFlow](https://github.com/PentesterFlow/agent),
built with **TanStack Router + Vite + React**.

- **Routing:** file-based TanStack Router (`src/routes/`), auto code-split per route.
- **Theme:** light / dark toggle using the **databuddy.cc** color tokens
  (neutral white/gray base with purple `#453c7c`, coral `#b74677`, amber `#e3a514` accents).
- **Type:** TypeScript, no UI framework — hand-written CSS in `src/styles.css`.

## Develop

```sh
npm install
npm run dev        # http://localhost:5173
npm run build      # production build → dist/
npm run preview    # serve the production build
```

## Structure

```
src/
  main.tsx              app entry + router
  styles.css            design system (databuddy palette, light + dark)
  data.ts               page content (skills, lifecycle, capabilities, …)
  routes/
    __root.tsx          shared nav + footer + theme toggle
    index.tsx           landing page (hero, lifecycle, install, security …)
    skills.tsx          /skills — all ten built-in skills
    security.tsx        /security — the security model
  components/
    Brand.tsx           wordmark + GitHub icon
    Terminal.tsx        self-typing hero terminal demo
    Install.tsx         tabbed install snippets with copy-to-clipboard
    Footer.tsx          site footer
    Reveal.tsx          scroll-reveal wrapper (IntersectionObserver)
    theme.ts            light/dark theme hook (persists to localStorage)
public/
  logo.png              PentesterFlow mark
```

The `routeTree.gen.ts` file is generated by the TanStack Router Vite plugin on
`dev`/`build` — don't edit it by hand.
